# To My 21 Years

**Interactive personal website documenting 21 years of life memories.**

Each year is a unique exhibit with text, photos, audio, and interactive elements. The site features:

- Timeline navigation
- Colorful themes for each age
- Memory wall for visitor messages
- Hidden chapters for family (Mom, Dad, Brother)
